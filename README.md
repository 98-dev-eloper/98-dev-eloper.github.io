# DNAEditor

DNA 시퀀스를 시각화하고 편집할 수 있는 오픈소스 벡터 에디터입니다.

## 주요 기능

- **DNA/AA 시퀀스 뷰어**: DNA 및 아미노산 시퀀스를 직관적으로 시각화
- **시퀀스 편집**: DNA 시퀀스 수정 및 조작 기능
- **다양한 파일 포맷 지원**: GenBank, FASTA, JSON, SnapGene, Geneious, JBEI-XML 형식 파싱 및 변환
- **생물학적 시퀀스 유틸리티**: DNA/RNA/단백질 시퀀스 처리를 위한 다양한 유틸리티 함수
- **범위(Range) 유틸리티**: 생물학적 범위 데이터 처리 기능
